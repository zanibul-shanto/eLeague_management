#Esports League Mangement
The following actions are included in the creation of this new system, which aim to create an online application while keeping the complete process in mind.
Before you continue, ensure you have met the following requirements:
    1. Install XAMPP 
    2. Open XAMPP Control panel and start apache and [MySQL] 
    3. Download the project
    4. Extract files in C:\xampp\htdocs & paste it here
    5. Open link localhost/phpmyadmin.
    6. Give a database name as (E_League) hit on create button.
    7. After creating database name click on import.
    8. Browse the file in directory [online-shopping/database/stop-get-furnish.sql].
    9. Open any browser and type "http://localhost/E_League/index.php”

You can show our users team & ADMIN gets its email and password to access their account.

Administrator of Esports League Management has multiple features such as Add, Delete, Update user’s information.

Features of Esports Management League:
    1.Secure registration and profile management facilities for e-sports athlets.
    2.Browsing through the emails to see the registered team that are already registered.
    3.Admin can login & logout with their email & password.

Back end: PHP, MySQL
    1.PHP: Hypertext Preprocessor (PHP) is a technology that allows software developers to create dynamically generated web pages, in HTML, XML, or other document types, as per client request. PHP is open source software.
    2.MySQL: MySQL is a database, widely used for accessing querying, updating, and managing data in databases.
